2. Required Software

Some software components need to be installed before using this instrument driver. The minimum versions of these components are listed below, and can be downloaded from the Download Site.

VISA 4.1 or later
LabVIEW 8.5 or later

Refer to the LabVIEW Help for more information about software requirements. You access the LabVIEW Help by selecting Help»Search the LabVIEW Help.

3. Known Issues

To report issues or provide feedback about this instrument driver, please send an email to admin@voltech.com.cn.

By the time of this driver release, this driver does not implement:
  • functionality of displaying and harmonic bar chart for voltage and current
  • functionality of displaying voltage, current and power waveforms.
  • functionality for start or stop integration process for "Clock Start Method" (in which a clock setting up action will be needed)
  • functionality to display the integration graph for "Integrator" operation mode
  • functionality for internal system clock user setup
  • functionality to display 4 results or 14 results screen on its LCD display

    For the above not implemented functions, Voltech Instruments provides IEEE 488.2 compatible commands for all of them. Please refer to "PM1000+ User Manual" if you want to know how to take advantage of the functions described above. "PM1000+ User Manual" can be downloaded at www.voltech.com
    When constructing your own program, and requesting large quantity of data from PM1000+ use the Close.vi provided with this driver. Otherwise it will at times cause the PM1000+ not to respond to any subsequent commands. The Close.vi will make sure to close the opening connection to PM1000+ properly and prepare the next successful connection for you. .
